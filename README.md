 Sobre o Projeto

Este projeto tem como objetivo apresentar de forma didática e interativa as características científicas dos ruivos - uma minoria genética única que representa menos de 2% da população mundial.

  Estatísticas Principais:
< 2% da população mundial é ruiva natural
0.17% possuem cabelo ruivo E olhos azuis (combinação mais rara)
13% da população escocesa é ruiva (maior concentração mundial)
20% mais anestesia necessária para ruivos em procedimentos médicos

 Contexto Educacional:
Disciplina: Biologia (com integração de História e Geografia)
 Curso:Técnico em Informática
Objetivo:Desenvolver um site estático com conteúdo científico
Tecnologias: HTML5, CSS3, JavaScript
